# Setup Environment of Development

Typing hints are supported in this environment.

1. Ensure you have Conda installed.
2. Create the basic environment. `conda env create -f environment.yml`
3. Activate the environment. `conda activate REvoDesignReproduce`
4. Install PyMOL from the source. `pip install git+https://github.com/schrodinger/pymol-open-source.git`
5. Install the rest of packages using `pip install -r requirements.txt --force-reinstall` (twice) to continue the installation.
