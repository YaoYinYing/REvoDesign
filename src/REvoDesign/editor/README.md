# Why Monaco?
Monaco is a code editor that provides a rich set of features for editing code, including syntax highlighting, code completion, and code navigation. It is designed to be lightweight and fast, making it suitable for use in web applications.
Monaco is a much more powerful in syntax highlighting, code completion, and code navigation. It also has a rich set of features that make it suitable for use in web applications.

Configuration file editing is much more error-prone on normal text editors that don't support syntax highlighting and intention guides. Monaco provides a much more powerful and intuitive experience for editing configuration files in YAML.
