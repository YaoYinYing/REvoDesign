# User Guide

This section covers the core REvoDesign workflows in detail. Each page
focuses on a specific stage or component of the enzyme redesign pipeline.

## Pages

- **[Cluster Methods](cluster-methods.md)** — Advantages, shortcomings, and
  selection guidance for each of the four built-in clustering algorithms,
  plus instructions for implementing and registering your own method.

- **[OpenKinetics](openkinetics.md)** — How to configure and use the optional
  OpenKinetics API-based activity predictor to score mutant libraries.

See the [Getting Started](../getting-started.md) page for installation and
first-launch instructions. The [API Reference](../api/core.md) documents
every module and its public API.
