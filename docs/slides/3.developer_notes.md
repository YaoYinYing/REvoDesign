# Package to Go

The basic purpose of REvoDesign to connect any available novel protein design tools together and provide a unified interface for users via PyMOL molecular visualizing system.

---

# *U*nified *P*rogramming *I*nterface (UPI)
Providing a general interface for all modules that plays the same role in the whole pipeline.

---

## From Published to Connected
1. 😵‍💫**Published**: The published version of modules may not be ready to be used and connected. 
   - **Hard** to be installed
   - **Incompatible** with some devices
   - **Complicated** to be used for tasks
   - **Chaotic** storage management for large pretrained weight files
   - **Unreasonable** module arrangement for maintainance
---
## From Published to Connected
2. 😘**Connected**: The connected version of modules are ready to be installed and used.
   - **Easy** installation process
   - **Compatible** with a wide range of devices
   - **Simplified** usage via programming interface
   - **Unified** storage policy for pretrained weight files
   - **Efficient** module arrangement for maintenance
